Emathdisplaymode

Create mathformulas in "displaystyle"

This jQuery-plugin is used to show a mathematical formula in LaTeX-like displaystyle in both viewmode and editmode. It depends on jQuery and MathQuill. Optionally mqpanel.js can also be used.

Here are couple of examples, how to use jquery.emathdisplaymode.js.

Non-editable

Using emathdisplaymode to show existing formula

<form> <div id="display1"></div> </form> <script type="text/javascript"> jQuery('#display1').emathdisplaymode({"latex":"2x+4x-8x=-2x", "editable":false}); </script>

Editable

Using emathdisplaymode to edit a formula

<form> <div id="display2"></div> </form> <script type="text/javascript"> jQuery('#display2').emathdisplaymode({"latex":"2x+4x-8x=-2x", "editable":true}); </script>

Editable displaymode field triggers changed-event when the content changes and addnext-event when enter-key is pressed. The addnext-event will also get extra parameters ['emathdisplaymode', place], where 'emathdisplaymode' is the type of this element and place is jQuery-object with the containing DOM-element.

These events can be used in the containing system (ebook, Moodle, etc.) to trigger getting the data from emathdisplaymode and saving it or continuing editing with the next element.
